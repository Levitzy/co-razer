HTML Paragraphs

The HTML <p> element defines a paragraph.

A paragraph always starts on a new line, and browsers automatically add some white space (a margin) before and after a paragraph.

Example

<p>This is a paragraph.</p>
<p>This is another paragraph.</p>

HTML Display

You cannot be sure how HTML will be displayed. Large or small screens, and resized windows create different results.

With HTML, you cannot change the display by adding extra spaces or extra lines in your HTML code. The browser will automatically collapse extra spaces and ignore extra line breaks.

Example

<p>
This paragraph
contains a lot of lines
in the source code,
but the browser
ignores it.
</p>

<p>
This paragraph
contains         a lot of spaces
in the source         code,
but the        browser
ignores it.
</p>

HTML Horizontal Rules

The <hr> tag defines a thematic break in an HTML page, often shown as a horizontal rule. It separates content or introduces a change of topic.

Example

<h1>This is heading 1</h1>
<p>This is some text.</p>
<hr>
<h2>This is heading 2</h2>
<p>This is some other text.</p>
<hr>

The <hr> tag is an empty element (no closing tag).

HTML Line Breaks

The HTML <br> element defines a line break. Use it for a new line without starting a new paragraph.

Example

<p>This is<br>a paragraph<br>with line breaks.</p>

The <br> tag is an empty element (no closing tag).

The Poem Problem

This poem will display on a single line if written inside a normal paragraph, because the browser collapses whitespace.

Solution — The HTML <pre> Element

The HTML <pre> element defines preformatted text, preserving spaces and line breaks and using a fixed-width font.

Example

<pre>
  My Bonnie lies over the ocean.

  My Bonnie lies over the sea.

  My Bonnie lies over the ocean.

  Oh, bring back my Bonnie to me.
</pre>

HTML Tag Reference

TagDescription

<br>Inserts a single line break
<pre>Defines pre-formatted text
