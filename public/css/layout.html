CSS Layout Foundations

Modern CSS provides powerful layout systems that replace brittle hacks. Understanding the document flow, flexbox, and grid allows you to build responsive interfaces with minimal code.

Normal Flow

Browsers position elements in two primary directions by default: block-level elements stack vertically, and inline elements flow horizontally inside each block. Managing flow is the first step before introducing advanced layout tools.

  • Block formatting context: Elements like <div>, <article>, and headings expand to fill available width and push subsequent blocks downward.
  • Inline formatting context: Text, <span>, and inline links sit on the same line, wrapping automatically; line-height controls the vertical rhythm.
  • Positioning adjustments: Margins collapse between block siblings, while paddings do not. Floating should be reserved for text wrap scenarios.

Flexbox for One-Dimensional Layouts

Flexbox excels when you arrange items along a single axis (horizontal nav bars, vertical sidebars, button groups). It distributes space and aligns items without manual margins.

.toolbar {
  display: flex;
  align-items: center;
  gap: clamp(0.75rem, 2vw, 1.25rem);
}

.toolbar button:last-child {
  margin-left: auto; /* pushes final button to the edge */
}

Use flex-wrap when items need to wrap and flex shorthand on the children to control how they grow or shrink.

Grid for Two-Dimensional Layouts

CSS Grid manages rows and columns simultaneously. Define tracks, then place items by line numbers, spans, or named areas.

.dashboard {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(220px, 1fr));
  gap: clamp(1rem, 3vw, 2rem);
}

.dashboard .summary {
  grid-column: 1 / -1; /* full width on large screens */
}

Grid pairs well with container queries and provides fine-grained control for responsive designs without complex media queries.

When to Choose Which

Flexbox

Use for menus, toolbars, notification trays, card carousels, or aligning items in a single row or column.

Grid

Reach for grid when items need to align on both axes: dashboards, marketing sections, masonry-style galleries.

Flow & Positioning

Sometimes the default document flow with margins is all you need. Combine it with CSS logical properties for internationalisation.
