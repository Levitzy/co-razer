CSS Grid

Grid is a two-dimensional layout system made for expressive compositions. Define tracks and place items across rows and columns with pixel-perfect control.

Grid Containers

Enable a grid by declaring display: grid; (or inline-grid). Tracks are the rows and columns that items snap to, and they follow the document's writing mode so vertical layouts behave naturally.

.layout {
  display: grid;
  grid-template-columns: repeat(3, minmax(0, 1fr));
  gap: clamp(1rem, 4vw, 2.5rem);
}

Track Definitions

Use repeat(), minmax(), and fractional units (fr) to describe flexible tracks. Combine with auto-fit or auto-fill for responsive galleries. auto-fit collapses empty tracks, while auto-fill keeps them reserved so content can flow into them later.

.gallery {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(220px, 1fr));
  gap: 1.5rem;
}

Named lines ([content-start]) help you anchor items to semantic positions.

Implicit Tracks And Alignment

  • Implicit tracks: When content exceeds the explicit grid, the browser creates additional tracks. Tune their sizing with grid-auto-rows and grid-auto-columns.
  • Auto placement: grid-auto-flow controls how new items fill the grid. Switch to column or dense to repack gaps.
  • Alignment: justify-items and align-items align content within each cell, while justify-content and align-content adjust the entire grid inside its container.

Named lines ([main-start]) and template areas keep placement readable and make large layouts easier to maintain.

Placing Items

Grid items can span multiple tracks. Use the grid-column and grid-row shorthands or the more descriptive grid-area. Items that are not explicitly positioned follow the grid-auto-flow algorithm, which defaults to filling rows before columns.

.hero {
  grid-column: 1 / -1;
}

.sidebar {
  grid-column: 1 / span 1;
  grid-row: 2 / span 2;
}

Template areas provide a declarative approach:

.dashboard {
  display: grid;
  grid-template-areas:
    "title title actions"
    "nav   main   main"
    "nav   footer footer";
}
