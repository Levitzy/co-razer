CSS Flexbox

Flexbox lays out elements along a single axis. It excels at nav bars, cards, toolbars, and any layout where space distribution matters.

Flex Containers

Once you declare display: flex;, the container gains a main axis (row by default) and a cross axis (column). Direct children become flex items that can shrink, grow, and align themselves. Use flex-direction to swap axes and row-reverse or column-reverse to flip ordering.

.toolbar {
  display: flex;
  align-items: center;

Aligning Items

Use justify-content to distribute space along the main axis and align-items for the cross axis. Combine with gap to space items evenly without margin hacks. When items wrap onto multiple lines, align-content controls how those lines distribute along the cross axis.

.pricing-row {
  display: flex;
  justify-content: space-between;
  align-items: flex-start;
  gap: clamp(1rem, 4vw, 2.5rem);
}

Switch to columns with flex-direction: column; when stacking on smaller screens.

Flexible Children

The flex shorthand controls growth (flex-grow), shrink (flex-shrink), and basis (initial size). Giving each card flex: 1 1 240px; lets them wrap responsibly. Remember that flex-basis overrides width, and setting min-width: 0 on flex children avoids unexpected overflow when content is wide.

.feature-card {
  flex: 1 1 240px;
  min-height: 200px;
}

Wrap items with flex-wrap: wrap; to prevent overflow. Individual items can override alignment with align-self or change visual order with order.

Patterns And Tips

  • Equal-height columns: Flex items stretch to the tallest sibling when align-items: stretch; (the default) is active.
  • Spacing push: Add margin-inline-start: auto; to a flex item to push it to the far edge, a common trick for CTA buttons in nav bars.
  • Vertical centering: In a column layout, justify-content: center; vertically centers items relative to the container height.
  • Fallbacks: For very old browsers that lack flexbox, provide a simple stacked layout via the normal flow and enhance with flex when supported.

Flexbox is efficient for UI controls and linear content. Reach for grid when alignment must occur in both dimensions.
